---- Installing

1. Need Composer and Node JS Intsalled
2. Recommended php > 8.2

* Download zip file and extract into your web development server folder
* Run "composer install"
* Then Run "npm install"
* Then Run "npm run dev"
* Run "php artisan serve" in another new terminal

You are good to go. Happy Coding...
